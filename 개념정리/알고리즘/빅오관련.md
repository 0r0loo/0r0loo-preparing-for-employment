# 시간 복잡도

범위를 보고 시간 복잡도가 얼마나 걸릴지 생각해보고 그에 맞는 알고리즘을 적용하자!



## 숫자

- 1,000 : 천

- 1,000,000 : 100만 (0 * 6개)

- 1,000,000,000 : 10억  (0 * 9개)

1억 => 1초

2^17 = 13만



# 빅오

## 1초가 걸리는 범위

- O(1)
- O(logN)
- O(N) : 1억 (100,000,000)
- O(NlogN) : 5백만 (5,000,000)
- O(N^2) : 1만 
- O(N^3) : 500
- O(2^N) : 27 (20~25)
- O(N!) : 11 (10)
